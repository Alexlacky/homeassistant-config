# dmatik’s Home Assistant configuration

I’m using Lovelace mostly for my mobile devices and HaDashboard for the tablet on the wall. 

## Hardware
- Broadlink RM Pro+
- Sessoo RF light switches
- Logitech Harmony Hub
- Switcher V2
- Yeelight Ceiling
- Sensibo
- Synology NAS
